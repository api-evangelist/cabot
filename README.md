# Cabot Corporation (cabot)
Cabot Corporation is a global specialty chemicals and performance materials company serving the transportation, infrastructure, environment, and consumer industries. The company's products include reinforcing carbons, specialty carbons, fumed silica, inkjet colorants, and engineered elastomer composites.
